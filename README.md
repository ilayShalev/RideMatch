# RideMatch - מערכת הסעות חכמה

מערכת להקצאת נהגים לנוסעים באמצעות אלגוריתם גנטי.

## תיאור הפרויקט

RideMatch היא מערכת חכמה לניהול הסעות המבוססת על אלגוריתם גנטי. המערכת מאפשרת שיבוץ אופטימלי של נוסעים לרכבים תוך התחשבות במגבלות קיבולת, זמני הגעה, ומרחקים.

### תכונות עיקריות

- שיבוץ אוטומטי של נוסעים לרכבים
- התחשבות בקיבולת הרכבים
- חישוב זמני איסוף אופטימליים
- ממשק משתמש נוח לניהול נהגים ונוסעים
- תמיכה בעדכון זמינות יומי

## דרישות מערכת

- Windows 10 ומעלה
- Visual Studio 2019 ומעלה
- .NET Framework 4.7.2
- SQL Server LocalDB

## התקנה

1. שכפל את המאגר
2. פתח את הפתרון ב-Visual Studio
3. התקן את חבילות NuGet הנדרשות
4. בנה והרץ את הפרויקט

## מבנה הפרויקט

- `Models/` - מודלים של הנתונים
- `Services/` - שירותים לוגיים
- `Forms/` - טפסי הממשק המשתמש
- `Database/` - קבצי מסד הנתונים

## שימוש במערכת

1. הוסף נהגים ונוסעים למערכת
2. הגדר את זמינות הנהגים והנוסעים ליום הבא
3. הפעל את האלגוריתם הגנטי
4. צפה בתוצאות השיבוץ

## פיתוח

הפרויקט משתמש באלגוריתם גנטי עם הפרמטרים הבאים:
- גודל אוכלוסייה: 100
- מספר דורות מקסימלי: 1000
- שיעור מוטציה: 1%
- אחוז אליטיזם: 20%
